# 🏋️ AI Gym & Fitness Assistant

## Overview

AI Gym & Fitness Assistant is an AI-powered fitness management platform developed as a Major Project for the AI Skill Development Course. The application combines workout tracking, personalized diet planning, habit monitoring, analytics, and conversational AI into a single integrated system.

The platform helps users achieve their fitness goals through intelligent recommendations, performance tracking, and personalized guidance.

## Installation

### Clone Repository

```bash
git clone <repository-url>
cd AI-Gym-Assistant
```

### Backend Setup

```bash
cd backend

python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt

python -m uvicorn app.main:app --reload
```

Backend URL:

```text
http://127.0.0.1:8000
```

API Documentation:

```text
http://127.0.0.1:8000/docs
```

### Frontend Setup

```bash
cd frontend

npm install

npm start
```

Frontend URL:

```text
http://localhost:3000
```
